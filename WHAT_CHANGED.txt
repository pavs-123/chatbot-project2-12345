🚀 HOW TO RUN (3 WAYS):

METHOD 1 - EASIEST (RECOMMENDED):
  Windows:    Double-click "RUN_CHATBOT.bat"
  Linux/Mac:  bash RUN_CHATBOT.sh
  
  → Automatically installs everything and starts server!

METHOD 2 - PYTHON LAUNCHER:
  python run_chatbot.py
  
  → Same auto-install, cross-platform

METHOD 3 - MANUAL (if already set up):
  uvicorn chatbot.api:app --reload --port 8001
  
  → Direct server start (fastest)

THEN:
  Open browser: http://localhost:8001
